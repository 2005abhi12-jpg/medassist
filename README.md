# 💊 MedAssist - Smart Medication Reminder System

MedAssist is an AI-powered medication management system designed to help patients take medicines on time and alert caregivers in real-time.

---

## 🚀 Features

- 🎤 Voice-controlled medication assistant
- ⏰ Smart reminders & alerts
- 📊 Adherence tracking (Taken / Missed / Snoozed)
- 👨‍⚕️ Caregiver notification system
- 📡 Real-time updates using WebSockets
- 🔔 Alert system for missed doses

---

## 🧠 Tech Stack

### Frontend
- React.js
- CSS (Custom UI)
- Web Speech API (Voice)

### Backend
- Node.js
- Express.js
- MongoDB

### Other
- Socket.IO (Real-time updates)
- Twilio (SMS alerts - optional)
